## Sobre o projeto

O **TaskFlow** é uma landing page criada para simular a apresentação de uma solução digital voltada para **produtividade e organização de tarefas**.

O projeto foi desenvolvido como peça de portfólio para demonstrar domínio em **estruturação semântica**, **estilização moderna**, **responsividade**, **interatividade com JavaScript** e construção de interfaces com aparência mais próxima de um produto real.

Nesta versão mais atualizada, a landing page recebeu melhorias visuais e técnicas importantes, tornando-se mais forte tanto para uso acadêmico quanto para apresentação profissional.

## Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar e demonstrar conceitos essenciais do desenvolvimento front-end, incluindo:

- estruturação semântica com HTML;
- estilização moderna com CSS;
- responsividade para diferentes dispositivos;
- manipulação do DOM com JavaScript;
- animações e interatividade;
- validação de formulário;
- alternância de tema;
- persistência de preferências com `localStorage`;
- organização visual orientada à experiência do usuário.

---

## Funcionalidades

- layout moderno e responsivo;
- navegação com menu mobile;
- destaque automático da seção ativa no menu;
- hero section mais forte e profissional;
- painel visual com métricas e fluxo de tarefas;
- cards de recursos com melhor hierarquia visual;
- seção “Sobre o projeto” mais bem estruturada;
- seção de destaques;
- seção de depoimentos simulados;
- formulário com validação aprimorada;
- feedback visual dinâmico no envio da mensagem;
- animações suaves com `IntersectionObserver`;
- barra de progresso animada;
- contador animado de usuários ativos;
- botão de retorno ao topo;
- modo escuro/claro com persistência no navegador.

1. Clone este repositório:
```
git clone https://github.com/phmbezerra/taskflow-landing-page.git
```
2. Acesse a pasta do projeto:
```
cd taskflow-landing-page
```
3. Abra o arquivo index.html no navegador.
