# TaskFlow - Landing Page

Uma landing page responsiva desenvolvida com foco em organização visual, usabilidade e interatividade.  
O projeto simula a apresentação de uma solução digital voltada para produtividade e organização de tarefas.

## Acesso ao projeto
🔗 [Clique aqui para acessar](https://phmbezerra.github.io/taskflow-landing-page/)

## Funcionalidades
- Layout moderno e responsivo
- Estrutura semântica em HTML
- Estilização com CSS
- Interatividade com JavaScript
- Contador animado de métricas
- Formulário com validação básica

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript

## Objetivo do projeto
Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais do desenvolvimento web, incluindo:
- estruturação de páginas com HTML
- estilização e responsividade com CSS
- interatividade com JavaScript
- organização de interface para melhor experiência do usuário

## Como executar localmente
1. Baixe ou clone este repositório
2. Abra a pasta do projeto
3. Execute o arquivo `index.html` no navegador

## Aprendizados
Durante o desenvolvimento deste projeto, foram praticados conceitos como:
- criação de layouts responsivos
- organização de conteúdo em landing pages
- manipulação básica do DOM
